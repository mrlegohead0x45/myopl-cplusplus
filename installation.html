Installation¶

Prerequisites¶

  • Have vcpkg installed

  • Have CMake installed

  • Have a C++ compiler

Compile¶

  1. Obtain the source code for this project from the releases page as the main branch is often a work in progress and may not even compile. Alternatively, if you want to help develop this project, see Development, as help is always welcome

  2. Create a build directory
    mkdir build; cd build
  3. Run CMake to generate build files, passing in the path to vcpkg
    cmake .. -DCMAKE_TOOLCHAIN_FILE=path/to/vcpkg/scripts/buildsystems/vcpkg.cmake.
    If you are cross-compiling (i.e compiling with a compiler not native to your system), you may have to pass -DVCPKG_TARGET_TRIPLET=triplet, where triplet is the triplet for your system
  4. Run those build files
    cmake --build .
