Development¶

If you want to help develop this project, great!

Get started by downloading the latest source code from the GitHub repository. You should also meet the same Prerequisites as for installation

You may want to add build/vcpkg_installed/triplet/include to your IDE’s include search path, as this is where installed header files will end up. triplet should be your vcpkg target triplet (see Triplets)

If you think you have made an improvement, no matter how small, please submit your changes as a pull request.

Please run clang-format with --style=file (to specify the options in .clang-format) on your code before you submit your pull request, however.
